# SwiftRide Transport System

SwiftRide is a comprehensive public transport management system built with Next.js, MongoDB, Stripe, and Google Maps.

## Features

- User authentication with role-based access (Admin, Driver, Passenger)
- Real-time bus tracking with Google Maps
- Booking system for passengers
- Route management for administrators
- Driver assignment and location updates
- Payment processing with Stripe
- Reporting and analytics

## Prerequisites

- Node.js 18+ and npm
- MongoDB database
- Stripe account
- Google Maps API key
